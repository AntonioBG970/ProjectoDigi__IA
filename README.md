=== INSTRUCCIONES ===

Para poder utilizarlo debes tener instalado lo siguiente:

- Python 3.12 (o posterior)
- PIP 26.1 (o posterior)
- Scikit-learn

Si no, te dará errores y no podrás ejecutarlo por falta de paquetes.

Estos son los comandos que debes escribir para instalar cada uno.

- Python (PIP incluido):

winget install Python.Python.3 (o en su página web https://www.python.org/downloads/)


- Scikit-learn

pip install scikit-learn

Para comprobar si se han instalado escribe esto:

python --version

pip --version
